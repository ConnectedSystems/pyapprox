get_morris_trajectory¶

pyapprox.sensitivity_analysis.get_morris_trajectory(nvars, nlevels, eps=0)[source]¶

Compute a morris trajectory used to compute elementary effects

Parameters
nvarsinteger

The number of variables

nlevelsinteger

The number of levels used for to define the morris grid.

epsfloat

Set grid used defining the morris trajectory to [eps,1-eps]. This is needed when mapping the morris trajectories using inverse CDFs of unbounded variables

Returns
trajectorynp.ndarray (nvars,nvars+1)

The Morris trajectory which consists of nvars+1 samples
